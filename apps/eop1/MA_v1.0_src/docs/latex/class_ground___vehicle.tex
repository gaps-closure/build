\doxysubsection{Detailed Description}
– Class responsible for generating a new ground mover for detections and updating it every second 

\begin{DoxyAuthor}{Author}
– Original Author -\/ Christopher Seay 

– Last Edited By -\/ Mases Krikorian 
\end{DoxyAuthor}
\begin{DoxyDate}{Date}
– 8/31/2020
\end{DoxyDate}
– This process is not part of MA and does not need a heartbeat to start. This process will not stop or terminate unless forced by the user. It generates a new moving object (near San Diego by default) for detection for \mbox{\hyperlink{class_e_o_i_r}{E\+O\+IR}} and \mbox{\hyperlink{class_r_d_r}{R\+DR}}, and updates its location every 1 second.
