Detailed Description

– Class responsible for processing (and possibly generating) video input to OpenCV and storing the detections; awaiting for a request from the sensor manager for detections and returning them.

– EOIR currently starts processing a video file that comes with the package. It stores the detections and awaits a request from the sensor manager (ISRM) for detections and combines the data from ground movers; adding the finding from the video (classification and confidence) to the detection list and sending it back to ISRM
