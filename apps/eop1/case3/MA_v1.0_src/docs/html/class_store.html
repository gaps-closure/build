Detailed Description

template<typename T>
class Store< T >

– Class responsible for all uses of memory of any generic data type as storage in the Mission Application Contents must be pointers.

Author
– Original Author - Mases Krikorian
– Last Edited By - Frederick Santiago
Date
– 9/17/2020

– MA Store uses tsl's ordered map implementation where insertion matters. Just like a regular map, the data can be access using any data type (such as a string), but the order of insertion is kept and can be accessed as an array using an integer to index the data location. Every entry must be a pointer to an object.
