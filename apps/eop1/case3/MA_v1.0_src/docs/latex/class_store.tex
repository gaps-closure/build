\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename T$>$\newline
class Store$<$ T $>$}

– Class responsible for all uses of memory of any generic data type as storage in the Mission Application Contents must be pointers. 

\begin{DoxyAuthor}{Author}
– Original Author -\/ Mases Krikorian 

– Last Edited By -\/ Frederick Santiago 
\end{DoxyAuthor}
\begin{DoxyDate}{Date}
– 9/17/2020
\end{DoxyDate}
– MA \mbox{\hyperlink{class_store}{Store}} uses tsl\textquotesingle{}s ordered map implementation where insertion matters. Just like a regular map, the data can be access using any data type (such as a string), but the order of insertion is kept and can be accessed as an array using an integer to index the data location. Every entry must be a pointer to an object.
