\doxysubsection{Detailed Description}
– Class responsible for providing useful functions that can be used by all systems. 

\begin{DoxyAuthor}{Author}
– Original Author -\/ Mases Krikorian 

– Last Edited By -\/ Frederick Santiago 
\end{DoxyAuthor}
\begin{DoxyDate}{Date}
– 9/15/2020
\end{DoxyDate}
– \mbox{\hyperlink{class_utils}{Utils}} provides various methods that can be used by all systems. Functions that include putting a thread to sleep, various unit conversions and other useful mathematical calculations, handling mission plan, and any other logic that can be useful for more than one system. All functions are provided statically.
